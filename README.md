# obsidian-plugin-near-kit
a plugin for obsidian that adds near blockchain interaction for function views, calls and transactions

ℹ️ THIS NEAR PLUGIN SAVES PRIVATE KEYS TO OBSIDIAN BROWSER LOCAL STORAGE. IT WILL NOT BE SYNCED WITH YOUR VAULT. IT IS ACCESSIBLE TO ALL PLUGINS
<br/>
ℹ️ this plugin was make using https://kit.near.tools/ - https://github.com/r-near/near-kit

---

### DEV

```sh
bun i
bun run dev
bun run version
bun run build
bun run tsc -noEmit
```
